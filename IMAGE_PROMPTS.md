## General Image Guidelines

### Style Requirements:
- **Color Palette:** Strictly black, white, and blues (#000000, #FFFFFF, #3B82F6, #1E40AF, #0F172A)
- **Design Theme:** Wavy patterns inspired by logo, modern, minimalist
- **Aesthetic:** Professional fintech, clean, high-tech, trustworthy
- **No Gradients:** Use solid colors only (as per design requirements)

### Technical Specifications:
- **Format:** JPG or PNG
- **Quality:** High resolution (at least 2x the display size for retina displays)
- **Optimization:** Images should be optimized for web
- **Aspect Ratios:** Maintain consistent aspect ratios within each section

### Content Guidelines:
- **Tone:** Professional, trustworthy, modern
- **Avoid:** Cluttered designs, too many colors, busy backgrounds
- **Focus:** Clean, minimalist, modern fintech aesthetic
- **Inclusion:** Subtle wavy design elements where appropriate
